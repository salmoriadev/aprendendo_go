Cryptopals Set 1
=================

Coleção em Go com as soluções dos oito desafios do Set 1 do Cryptopals.

Pré-requisitos
--------------

- Go 1.22 ou superior.

Como executar
-------------

```bash
go run .
```

Arquitetura
-----------

- `main.go` executa cada desafio em sequência.
- `desafio0x/` contém a implementação específica de cada exercício e os arquivos de apoio fornecidos pelo enunciado.

Referências
-----------

- https://cryptopals.com/sets/1
